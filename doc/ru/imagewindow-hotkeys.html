Сочетания клавиш

В этом разделе описаны сочетания клавиш, которые Вы можете применять в окне просмотра изображения.

Сочетания клавиш для управления непосредственно окном

Esc, X, Return

Закрыть окно просмотра

Q

Панель быстрой навигации

Средний клик, F

Во весь экран

T

Панель инструментов

Z

показать меню с действиями масштабирования

S

Строка статуса с различной информацией об изображении

/

Показать диалог помощи по сочетаниям клавиш

Правый клик, M

Контекстное меню

N

Сменить фильтр - влияет на качество масштабированного изображения

Shift+левая кнопка мыши

Выделить регион для масштабирования

Сочетания клавиш для навигации по файлам

Пробел, PageDown

Следующее изображение - показать следующее изображение в текущем каталоге и выделить его в навигаторе

Backspace, PageUp

Предыдущее изображение - показать предыдущее изображение в текущем каталоге и выделить его в навигаторе

Home

Первое изображение - показать первое изображение в текущем каталоге и выделить его в навигаторе

End

Последнее изображение - показать последнее изображение в текущем каталоге и выделить его в навигаторе

Сочетания клавиш для передвижения изображения

Стрелки

Сдвиги влево, вправо, вверх, вниз

Ctrl+Стрелка влево

Повернуть влево

Ctrl+Стрелка вправо

Повернуть вправо

Ctrl+Стрелка вверх/вниз

Повернуть на 180 градусов вверх/вниз

Shift+Стрелка влево/вправо

Повернуть на 1 градус влево/вправо

Сочетания клавиш для масштабирования изображения

+/-

Масштабировать в N раз (задаётся в настройках)

Ctrl++/-

Масштабировать 2x/0.5x

Колёсико мыши

Загрузить следующее/предыдущее изображение ИЛИ масштабировать вверх/вниз (задаётся в настройках)

Shift+Колёсико мыши

Аналогично +/-

Ctrl+Колёсико мыши

Аналогично Ctrl++/-

1...9

Масштабировать в 1...9 раз

0

Масштабировать до 1:1

Запятая

Уместить изображение по ширине в окно

Точка

Уместить изображение по высоте в окно

*

Уместить изображение по ширине и высоте в окно

Сочетания клавиш для управления текущим изображением

V

Отразить зеркально по вертикали

H

Отразить зеркально по горизонтали

R

Обнулить значение сдвигов, масштибирования и поворта (т.е. показать изображение в исходном положении)

P

Показать свойства изображения и файла

L

Игнорировать масштабирование по ширине, высоте и обоим параметрам, если текущее изображение меньше окна

I

Показать меню со страницами изображения

A

Остановитьили стартовать анимацию, если изображение анимированное

B

Рисовать фон для прозрачных изображений

K

Рисовать уголки вокруг изображения

E

Покзать меню с внешними программами, с помощью которых Вы можете открыть текущее изображение

F1

Показать первую страницу в многостраничном изображении

F2

Показать предыдущую страницу в многостраничном изображении

F3

Показать следующую страницу в многостраничном изображении

F4

Показать последнюю страницу в многостраничном изображении
