Аргументы командной строки для KSquirrel

Команда запуска KSquirrel из командной строки выглядит так:

ksquirrel [-t] [-r] [-l] [файл]

Параметр -t заставляет KSquirrel запустить процедуру создания эскизов изображений для указанного каталога файл.

Параметр -r применяется вместе с -t, и указывает KSquirrel, что сканирование каталога должно происходить рукурсивно (т.е. и во всех подкаталогах).

Параметр -l указывает KSquirrel вывести на экран список найденных кодеков и выйти. Этот параметр чисто информационный.

Параметр файл должен быть путём к каталогу, когда задан параметр -t. Иначе это путь к файлу или каталогу для открытия.

Аргументы командной строки для ksquirrel-small

Команда запуска ksquirrel-small из командной строки выглядит так:

ksquirrel-small [файл]

Параметр файл должен быть путём к файлу для открытия.

Примеры

  • Рекурсивно создать эскизы изображений, находящихся в каталоге /home/me/images.

    ksquirrel -t -r /home/me/images/

  • Открыть при запуске изображение /home/me/images/foto00054.jpg

    ksquirrel /home/me/images/foto00054.jpg

  • Перейти при запуске в каталог /home/me/images

    ksquirrel /home/me/images

  • Открыть изображение /home/me/images/foto00054.jpg с помощью ksquirrel-small

    ksquirrel-small /home/me/images/foto00054.jpg
